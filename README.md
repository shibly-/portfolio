# Portfolio Website

A simple personal portfolio built with **React**, **Vite**, and **React Router**.

## Pages

- **Home** — Introduction and quick highlights
- **Technical Skills** — Languages, frontend, backend, and tools
- **Experience** — Work history timeline
- **Education** — Degrees and coursework
- **Interest** — Hobbies and passions
- **Contact** — Email, links, and contact form

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Customize

Edit the placeholder content in `src/pages/` and update your name in `Home.jsx` and the footer in `Layout.jsx`.

## Build for production

```bash
npm run build
npm run preview
```
